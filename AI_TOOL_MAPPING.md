### Example 1: Sports Service Date Bug (AI Error → AI Fix)

**Initial AI-Generated Code (WRONG):**
```javascript

        const targetDate = new Date();
        targetDate.setDate(targetDate.getDate() + daysAhead);
        const dateString = targetDate.toISOString().split('T')[0].replace(/-/g, '');
        
        const url = `${this.baseUrl}/${sportLeague}/scoreboard?dates=${dateString}`;
        const response = await axios.get(url);
        
        // Problem: Only looking for UPCOMING games (not started)

            return event.status.type.state === 'pre';  // ❌ Excludes live games!
        });

            return this.formatGames(upcomingGames, sport);
        }

```

**Human Feedback:** "The sports scores disappear when games start"

```javascript

    const response = await axios.get(url);

        return this.formatGames(response.data.events, sport);

}
```

### Example 2: Voice Command HTTP Method Bug (AI Error → AI Fix)

**Initial AI-Generated Code (WRONG):**
```python
def send_spotify_command(self, action):
    endpoints = {
        'play': '/api/spotify/play',
        'pause': '/api/spotify/pause',
    }
    
    url = f"http://backend:3001{endpoints[action]}"
    response = requests.post(url, json={}, timeout=2)  # ❌ Always POST
```

**Human Feedback:** "Play and pause commands return 403 errors"

**AI-Fixed Code (CORRECT):**
```python
def send_spotify_command(self, action):
    endpoints = {
        'play': {'method': 'PUT', 'url': '/api/spotify/play'},      # ✅ PUT
        'pause': {'method': 'PUT', 'url': '/api/spotify/pause'},    # ✅ PUT
    }
    
    endpoint = endpoints.get(action)
    method = endpoint['method']
    
    if method == 'PUT':
        response = requests.put(url, json={}, timeout=2)  # ✅ Correct method
    else:
        response = requests.post(url, json={}, timeout=2)
```

### Example 3: Word Boundary Bug (AI Error → AI Fix)

**Initial AI-Generated Code (WRONG):**
```python
def process_command(self, text):
    text_lower = text.lower()
    
    # Check home navigation
    for keyword in COMMANDS['home']:  # 'home': ['home', 'back', 'go back']
        if keyword in text_lower:  # ❌ "back" matches in "playback"
            self.send_page_command('home')
            return True
```

**Human Feedback:** "Saying 'playback' goes to home page instead of playing music"

**AI-Fixed Code (CORRECT):**
```python
def process_command(self, text):
    text_lower = text.lower()
    
    # Check playback commands FIRST
    for action in ['play', 'pause']:
        for keyword in COMMANDS[action]:
            if keyword in text_lower:
                self.send_spotify_command(action)
                return True
    
    # Check home with word boundaries
    for keyword in COMMANDS['home']:
        # ✅ Word boundary prevents "back" matching in "playback"
        if f" {keyword} " in f" {text_lower} " or text_lower == keyword:
            self.send_page_command('home')
            return True
```

## Development Workflow

### Typical Interaction Pattern

1. **Human:** "I need a service to fetch NBA scores from ESPN API"
2. **AI (Copilot/ChatGPT):** *Generates complete sports.js service with caching*
3. **Human:** *Tests the code*
4. **Human:** "The live game scores are missing"
5. **AI:** *Analyzes code, identifies filter issue, generates fix*
6. **Human:** *Tests fix* "That works now"

### Bug Fix Process

All bugs were fixed through this pattern:
1. Human observes issue and describes it to AI
2. AI requests logs or additional context
3. Human provides requested information
4. AI identifies root cause and generates fix
5. Human tests and confirms (or reports continued issues)
6. Repeat until resolved

- Static ETA not updating (missing live clock)
- HTTP method mismatches (POST vs PUT)
